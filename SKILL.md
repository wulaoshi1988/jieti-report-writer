# 课题研究报告写作

## 概述

将教育科研课题的结题材料撰写为规范的学术研究报告。核心原则：重点不是"做了哪些活动"，而是"为什么要研究、怎样开展研究、取得了什么成果"。默认使用学术表达、过去时/完成时，保证研究目标、内容、方法、成果一一对应。

## 参考模板

本技能附带完整的写作模板和规范，位于 `references/研究报告写作模板.md`。模板包含：

- 报告推荐结构（题目 → 摘要 → 关键词 → 正文十节 → 结尾三节）
- 每节的详细写法要求和禁忌
- 交稿前自查清单
- 可直接套用的 Markdown 骨架

**在任何写作步骤中，均以该模板为基准。** 遇到结构或写法的疑问时，优先查阅模板对应章节。

## 工作流程

### 第一步：信息采集

开始写作前，先向用户确认以下关键信息（如用户已提供则跳过对应问题）：

1. **课题基本信息**：课题名称、课题组、执笔人
2. **研究背景材料**：课题的研究背景（政策/学术/实践）、研究意义
3. **核心概念**：需要界定的术语、操作性定义
4. **研究设计**：研究目标（3-4个）、研究内容、研究方法（3-4种）
5. **研究过程**：阶段划分、时间节点、关键事件
6. **已有成果素材**：已发表的论文、形成的模式/策略、数据、案例等

一次性问 2-3 个问题，不要一次抛出所有问题，避免让用户感到压力。

### 第二步：生成大纲

根据用户提供的信息，生成完整报告大纲。大纲需严格遵循模板的推荐结构，逐节列出各节的要点和规划篇幅。将大纲展示给用户确认后再进入逐节写作。

**特别注意**：
- "研究成果、主要结论与观点"应占总篇幅约 50%
- 确保研究目标、内容、方法、成果之间存在清晰的对应链条

### 第三步：逐节写作

按照大纲顺序逐节撰写。每写完一节，展示给用户审阅，确认后再进入下一节。

**各节写作要点速查**（详细要求见模板）：

| 章节 | 关键要点 |
|------|---------|
| 摘要 | 300字左右，重成果和结论，含关键数据 |
| 关键词 | 名词性术语，2-5个 |
| 研究背景 | 从政策/学术/实践中选，说明问题来源和必要性 |
| 研究意义 | 理论意义 + 实践意义 |
| 课题界定 | 核心概念的操作性定义 |
| 研究目标 | 动词+名词，3-4个 |
| 研究内容 | 与目标对应，有逻辑顺序 |
| 研究方法 | 3-4种，说明"做什么""怎么做" |
| 研究过程 | 分阶段，含时间/地点/对象/措施 |
| 研究成果 | 全文重点，论据+论证，忌只列论文/奖项 |
| 研究效果 | 前后对比，量化优先 |
| 问题与设想 | 方法/样本局限，后续方向 |

### 第四步：质检自查

全文完成后，对照模板中的"自查清单"逐项检查，并输出检查结果：

- [ ] 是否清楚回答了"为什么研究、怎样研究、成果是什么"
- [ ] 是否把研究报告误写成工作报告
- [ ] 是否存在"流水账"写法
- [ ] 是否缺摘要、关键词、署名、参考文献、附录
- [ ] 研究目标、内容、方法、成果是否一一对应
- [ ] 是否有研究前后对比数据
- [ ] 是否存在大段复制、未标出处、整段雷同风险

对未通过项给出修改建议。

## 注意事项

- **学术语气**：全文使用学术表达，避免口语化、抒情化
- **时态一致**：默认使用过去时和完成时
- **区分报告类型**：研究报告 ≠ 工作报告，重点在"研究发现了什么"而非"开展了什么活动"
- **成果写作禁忌**：不要只列论文题目、获奖名称、活动次数——这些都是成果形式，不是成果本身。要写"数据 + 典型事例 + 具体描述"
- **文献引用**：引用他人观点/数据时使用脚注，参考文献格式参考 GB/T 7714-2015
- **附录原则**：少而精，只放对正文有支撑价值的材料
